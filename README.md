# clojure-start

Some clojure experiments

## Installation

```zsh
lein
```

## Usage

To run the clojure program:

```zsh
    $ lein run
```

To see the clojure script playground:

```shell
    $ lein cljsbuild auto
```

```shell
    $ cd target/resources/public && python -m SimpleHTTPServer
```
